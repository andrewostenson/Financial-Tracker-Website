# Financial-Tracker-Website
An introduction to personal budgeting built with HTML, JS, and PHP

# Features
- Dynamically generated expense table and pie chart
- User information stored in a database with hashed passwords
- Account registration, login, and deletion
- User preferences such as color theme

## Technologies
- AJAX/Restful API
- SQL
- PHP
- HTML
- CSS

## How to Run
1. Download XAMPP and run Apache and SQL servers
2. Place PBT folder in your htdocs folder on your C:/ drive
3. Type localhost/PBT/register.html into your search bar
